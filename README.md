## ccutil

this is c++ toolkit

### Usage

```shell
git clone git@github.com:haormj/ccutil.git
```

#### http library

```shell
cd ccutil/module/http
mkdir build
cd build
cmake ..
make 
make test
make install
```

#### vision library

```shell
cd ccutil/module/vision
mkdir build
cd build
cmake ..
make 
make test
make install
```
